Module: js/vr360.js → window.VR360

FeatureDetail
RendererThree.js WebGL, inverted sphere (r=500, 64 segments)
Panorama sourceProcedurally generated canvas (2048×1024)
NavigationMouse drag → lon/lat rotation
ZoomScroll wheel → FOV 40°–95°
Auto-rotateToggle, pauses on drag interaction
FullscreenNative Fullscreen API

User Flows

Primary: Browse & Book

Page Load └── Loading Screen (fetch JSON + Three.js init) ↓ ~2–3 seconds └── VR Experience (default: sky-lounge scene) ↓ user drags / clicks hotspots └── Explore Scenes ↓ user clicks "Xem Bảng Giá" └── Floorplan Overlay (filter → select unit → "Đặt Cọc") ↓ └── Booking Modal (pre-filled with unit data) ↓ fill name + phone → submit └── Success Screen → callback CTA

Loan Calculation

Open Loan Calculator ↓ adjust price slider ↓ set equity % + term + bank ↓ live output updates (monthly payment / total interest) ↓ view amortization chart ↓ click "Tư Vấn Vay Vốn" └── Booking Modal opens

Voice AI Chat

Click AI FAB → panel slides in ↓ demo bubbles appear ↓ user clicks mic └── Panel → capsule mode ↓ mic permission granted ↓ waveform appears, listening starts ↓ user speaks ↓ STT transcribes → user bubble ↓ bot generates reply → bot bubble ↓ TTS speaks reply └── auto re-listen (loop until mic clicked again)
